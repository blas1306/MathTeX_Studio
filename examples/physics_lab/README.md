# Physics Lab Report Demo

This demo shows one of the main goals of MathTeX:

> writing a technical document where calculations and plots are generated directly inside the same `.mtex` file.

## What it demonstrates

- defining measurement vectors
- computing summary values
- generating a plot
- injecting results into the final document with `\var{...}`
- compiling everything into a PDF

## Main idea

Instead of using:
- one tool for calculations,
- another for plots,
- and another for writing,

MathTeX keeps everything in one place.

## File

- `main.mtex` → executable LaTeX document
