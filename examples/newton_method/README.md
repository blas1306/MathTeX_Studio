# Newton Method Demo

This demo shows how MathTeX can be used to create a short numerical methods note.

## What it demonstrates

- defining a function
- computing a numerical root with Newton-Raphson
- embedding the computed result into the final document with `\var{...}`
- generating a plot from the same code block

## Main idea

Instead of computing the approximation in one tool and then copying the result into a document manually, MathTeX keeps the computation and the report together in the same `.mtex` file.

## File

- `main.mtex` → executable LaTeX document
