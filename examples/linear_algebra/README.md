# Linear Algebra Report Demo

This demo shows how MathTeX can be used to write a short technical report for a linear algebra problem.

## What it demonstrates

- matrix and vector definition
- solving a linear system \(Ax=b\)
- computing an LU factorization
- embedding all computed results directly into the final document with `\var{...}`

## Main idea

Instead of solving the system in one tool and then copying the results into another document editor, MathTeX keeps both the computation and the report in the same `.mtex` file.

## File

- `main.mtex` → executable LaTeX document
