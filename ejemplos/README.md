# Ejemplos de MathTeX

Esta carpeta concentra los archivos de ejemplo del proyecto.

- `documentos/`: ejemplos `.mtex` y sus artefactos asociados (`.tex`, `.pdf`, imagenes).
- `scripts_raiz/`: scripts y funciones que antes estaban en la raiz del repo.
- `temporales/`: borradores o salidas temporales que conviene mantener fuera de la raiz.

Los `.mtx` que ya estaban dentro de `ejemplos/` se conservaron ahi para no mezclar nombres que hoy colisionan entre variantes del mismo ejemplo.
